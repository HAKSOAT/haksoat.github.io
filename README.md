### HAKSOAT's Blog

Static site generated using Jekyll and hosted on GitHub Pages.

Hosted at: [haksoat.com](https://haksoat.com)

Update packages:
```
bundle update
```

Run locally:
```
bundle exec jekyll serve
```

Run locally, as though in production. Mainly to see things like analytics:

```
JEKYLL_ENV=production bundle exec jekyll serve --baseurl ""
```
